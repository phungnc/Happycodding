## JSX

JSX = JS + XML

[Detail of JSX](http://facebook.github.io/jsx/)

Sample

```js

var Hello = React.createClass({
  render: function() {
    return (
      <div>Hello {this.props.name}</div>
    );
  }
})

```

The element ```<div> ... </div>``` is JSX syntax.
